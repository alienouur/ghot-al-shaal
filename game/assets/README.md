# مجلد الأصول (Assets)

هذه اللعبة **لا تعتمد على أي ملفات أصول خارجية**:

- **الرسومات** (`/images`): كل الشخصيات والبيوت والأشجار والسيارات والمؤثرات
  مرسومة برمجيًا عبر Canvas API (أشكال، تدرجات، خطوط) داخل
  `js/player.js` و `js/world.js` و `js/enemy.js` و `js/particles.js`.

- **الأصوات** (`/sounds`) و**الموسيقى** (`/music`): كلها مولّدة لحظيًا عبر
  Web Audio API داخل `js/audio.js` (مذبذبات + ضجيج مفلتر).

يمكنك لاحقًا وضع ملفات صور/أصوات حقيقية هنا وتعديل الأنظمة لتحميلها،
دون الحاجة لتغيير بنية اللعبة.
