# غوط الشعال: شارعنا 🏘️

لعبة مغامرة 2D Side-Scrolling عربية كاملة، مبنية **من الصفر** بدون أي محرك ألعاب:
**HTML5 + CSS3 + JavaScript ES6 + Canvas API** فقط.

## التشغيل
- **محلي (Offline)**: افتح `index.html` مباشرة في أي متصفح حديث.
- **أونلاين (رومات)**: يحتاج Node.js —
```bash
npm install   # مرة واحدة فقط
npm start     # يشغّل السيرفر على المنفذ 8430
```
ثم افتح `http://localhost:8430` من جهازك، ومن الهواتف (نفس شبكة الواي فاي): `http://[IP-جهازك]:8430`
(لمعرفة الـ IP على ويندوز: `ipconfig` → IPv4 Address)

## الأطوار
- **طور القصة**: 5 فصول في شارع غوط الشعال — من يوم عادي حتى المواجهة النهائية مع نورالدين بوس (3 مراحل).
- **وضع شارعنا** (2-5 لاعبين محليًا): ملك الشارع، آخر واحد في الشارع، كنز الشارع، المطاردة، سباق الأزقة، الكرة الذهبية، الغميضة.
- **🌐 أونلاين** (2-5 لاعبين، كل واحد من جهازه/هاتفه): دير روم واحصل على كود من 4 حروف، أصحابك يدخلوا بالكود، وصاحب الروم يختار النمط ويبدأ. كل أنماط وضع شارعنا متاحة أونلاين.

## الشخصيات
| الشخصية | الدور | القدرة |

| علي | القائد المتوازن | النداء — تشتيت الأعداء |
| عاصم | السريع | الخطفة — اندفاع سريع |
| عبد العليم | القوي | الدفعة — دفع الأعداء |
| صديق | الماكر | العين — تجميد الخصم |
| علوني | الذكي | التمويه — اختفاء وسرعة |

## التحكم (لاعب 1)
- **الحركة**: A / D — **القفز**: مسافة — **الجري**: Shift
- **القدرة**: E — **الضربة**: F — **التسلق**: W/S على السلالم
- **إيقاف مؤقت**: Esc — بقية اللاعبين: انظر شاشة إعداد وضع شارعنا.

## البنية
```
/game
  index.html
  /css/style.css
  server.js  (سيرفر الأونلاين — HTTP + WebSocket رومات)
  /js   (16 وحدة: محرك، فيزياء، كاميرا، جزيئات، صوت، حفظ، شبكة، أونلاين، لاعب، أعداء، عالم، قصة، جماعي، واجهة...)
  /data (ملفات JSON مرجعية — البيانات مضمنة في js/data.js لتعمل من file:// مباشرة)
  /assets (كل الرسوم والأصوات مولّدة برمجيًا)
```

## تقنيًا
- رسوم كرتونية مولّدة بالكامل بأشكال Canvas وتدرجات.
- أصوات وموسيقى مولّدة عبر Web Audio API.
- حفظ التقدم والإنجازات والإعدادات في localStorage.
- أونلاين: سيرفر Node.js + ws — الهوست يدير منطق النمط ويبث لقطات الحالة، وكل لاعب يتحكم بشخصيته من جهازه (20 تحديث/ثانية).
- دعم كامل للهواتف: أزرار لمس + تلميح تدوير الشاشة.
- ذكاء اصطناعي بآلة حالات: IDLE / PATROL / CHASE / ATTACK / STUN / RETURN.
- كاميرا بتتبع سلس، زوم جماعي، اهتزاز، ولقطات سينمائية.
- Culling للرسم خارج الشاشة وحد أقصى للجزيئات للأداء على الأجهزة الضعيفة.
